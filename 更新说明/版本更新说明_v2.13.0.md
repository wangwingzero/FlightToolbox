## 🧮 飞行计算工具总入口

### 统一入口，一屏找齐

- 新增首页入口「飞行计算工具」，把原本分散在各处的计算器集中起来
- 按功能分为三大类：
  - **飞行几何**：五边高度、下降率、梯度计算、转弯半径
  - **单位换算**：距离 / 速度 / 重量等常用单位互换
  - **天气相关**：侧风分量、气压换算、温度换算、ISA 温度
- 顶部分类标签支持横向滑动，方便在「全部 / 飞行几何 / 单位换算 / 天气」之间快速切换

### 常用场景示例

- 进近前估算「某点大概多少高度」：用 **五边高度** 或 **梯度计算**
- 起飞/落地前快速看侧风大小：用 **侧风分量**
- 跨资料/跨机队时遇到不同单位：用 **距离/速度/重量换算**
- 需要看 ISA 偏差、温度换算或 QNH/QFE 之间切换时，也都在这一页集中入口

## 📚 术语中心：缩写、定义与 IOSA 一站式查询

### 三类术语统一查询

- 新增「术语中心」专用页面，将原本分散在文档里的信息整理为可搜索的术语库：
  - **缩写（Abbreviation）**：英文缩写 + 中文翻译 + 英文全称
  - **权威定义（Definition）**：中文名称 + 英文名称 + 正文定义
  - **IOSA 审计相关术语（IOSA）**：中文/英文名称 + 定义 + 等效术语/参见
- 顶部提供「全部 / 缩写 / 权威定义 / IOSA 审计」四个标签，可随时切换视图

### 智能搜索与详情弹窗

- 支持按 **缩写、中文名称、英文名称、定义内容** 等多字段搜索
- 搜索结果以卡片形式展示：左侧类型徽章（AB/DEF/IOSA），右侧显示来源信息
- 点击卡片后，从底部弹出详情弹窗：
  - 对缩写：展示缩写、英文全称、中文翻译
  - 对权威定义：突出中文/英文名称，并高亮正文中出现的其他术语，方便「顺藤摸瓜」
  - 对 IOSA 术语：除定义外，还列出「等效术语 / 参见」等，方便对照阅读
- 无结果时会给出友好的空状态提示，并引导缩短关键字或切换类别

### 用户能获得什么

- 飞行员和教员可以在一个地方快速查到各类术语的 **缩写、全称与标准定义**
- IOSA 相关术语与等效词一并呈现，方便准备审计或做内部培训
- 与体检标准、规章中心等模块配合，更容易统一术语理解

## 📡 通信技术要点与标准话语页面焕新

### 通信技术要点（通信规则页重构）

- 原「通信规则」页面完全重构为「通信技术要点」，不再需要层层点击折叠
- 页面按主题分区直接铺开，例如：
  - **通话概述 / 通话结构 / 通话技巧**
  - **数字、高度、时间、气压、航向、频率、跑道、速度的标准读法**
- 采用类似 iOS 的分组卡片与双色网格布局：
  - 常用数字、高度、航向、跑道等以 **网格形式** 展示，一眼扫过就能找到
  - 通话技巧、时间/气压/频率读法等以 **左侧彩色条卡片** 展示，便于记忆

### 陆空通话规范与标准话语

- 「标准话语」页面 UI 全面升级：
  - 顶部提供类似 iOS 分段控制的模式切换，可在「陆空通话」与「技术要点」之间一键切换
  - 搜索框、分类标签和卡片样式统一采用苹果风格设计，更贴近日常 iOS 体验
- 短语卡片突出：
  - 英文蓝色斜体、中文常规字体，方便快速对照
  - 分类与使用场景通过彩色徽章标明，例如起飞、进近、地面滑行等

### 用户能获得什么

- 不用再翻纸质资料或长文档，在小程序里即可 **快速对照标准读法**
- 进近前、签派沟通或检查单准备时，更容易找到需要的句型和读法
- 陆空通话相关内容在视觉上更清晰，适合飞行前快速「扫一眼」复习

## 🔄 与 v2.12.0 的差异总结

- v2.12.0：重点在 **CCAR 规章中心** 与 **「空勤灶」膳食指南**，为法规与健康管理打基础
- v2.13.0：重点在 **飞行计算与术语查询体验**，同时对多页面 UI 做了系统性升级

从对比角度看：

- v2.12.0 更偏向「内容扩展」：新增规章与膳食数据模块
- v2.13.0 更偏向「入口与界面升级」：
  - 把飞行计算器做成一站式入口
  - 把术语查询独立成「术语中心」
  - 将通信、话语、机场数据、体检标准、胜任力、IOSA 等模块的界面统一为更易读的 iOS 风格
