# 1 目的和范围

为规范上海飞行部事件信息的报告、收集、分析和应用，实现安全信息共享，推进安全管理体系建设，及时管控飞行运行中的安全风险，发现和消除安全隐患，预防影响飞行安全的情况发生，根据国家法律、民航法规和公司规章制度本规定。

本规定明确了事件信息报告的报告范畴、报告时机、报告方法、填报规范、上报流程、处理规范和事件信息报告工作的相关定义。

本规定适用于所有涉及中国东方航空股份有限公司上海飞行部的事件信息报告和填报工作。

# 3 术语和定义

下列术语和定义适用于本规定

# 3.1 事件信息

3.1.1 本规定所称事件信息，是指在民用航空器运行阶段或者机场活动区内发生航空器损伤、人员伤亡或者其他影响航空器安全运行的情况。主要包括：民用航空器事故、民用航空器征候以及民用航空器一般事件信息。  
3.1.2 一般事件是指在运行阶段或者机场活动区内发生人员受伤、造成直接经济损失或者其他影响航空器安全运行的情况，但其严重程度未构成生产安全事故、民用航空器征候的事件，包括但不限于公司航空安全差错、运行偏差、运行不正常动态和航空器非预期状态等。本规定所称一般事件信息不包括诸如飞行品质监控类3级事件、流控延误、配餐保障等不直接影响航空器安全运行的情况。  
3.1.3 影响航空器安全运行的情况，包括但不限于导致或可能造成：航空器中断起飞、采取避让措施、空中等待、改变进近方式、中止进近、复飞、返航、备降、跑道侵入、占用跑道、实施跑道检查、紧急制动、航空器损伤等。  
3.1.4 敏感事件包括且不限于民用航空器事故、民用航空器征候、《民用航空安全信息管理规定》定义和《事件样例》中列举的紧急事件、符合公司航空安全差错标准中严重差错情形的事件、符合公司安全红线标准的事件、已受到民航监管部门调查问询的事件、已被运行相关单位（机场、管制、维护保障、运行控制中心等）质询或上报的事件。

# 3.1.5 相关事件标准

a) 事故参照《民用航空器事件调查规定》的定义;  
b) 征候参照《民用航空器征候等级划分办法》的定义和标准;  
c) 严重差错参照《中国东方航空股份有限公司航空安全差错标准》的定义和标准；  
d) 安全红线参照局方规定的“安全红线”和《中国东方航空股份有限公司航空安全奖惩管理办法》中“安全红线”标准。

# 3.2 事件信息报告

# 3.2.1 事件信息首报

是指敏感事件发生后，由责任机长或事发单位，在可能的情况下立即通过电话向上海飞行部安全管理部报告事件的事实和现象等信息，由飞行部安全管理部对事件信息进行初步分析和判断，以及时、正确地启动敏感事件响应机制。事件信息首报规定适用于发生敏感事件的情况，以通过电话联系安全管理部信息值班人员通报情况的方式执行。

# 3.2.2 事件信息初报

是指事发相关单位在事件发生后，督促指导涉及事件的航班机长，及时按标准格式和规定要素，如实规范地完成事件信息填报，并按需提交并妥善保存与事件有关的文本、影音、数据以及其他资料，上报飞行部安全管理部对事件信息进行分析和判断，由飞行部安全管理部按照分级管理原则，依据安全信息管理规定对初报信息进行处理。

# 3.2.3 事件信息续报

是指已完成事件信息初报后，责任机长个人对事件信息存在补充需求以及事发单位获得新的事件情况时，主动补充填报并更新所提交的事件信息初报；或按照局方、公司以及飞行部安全管理部对事件信息的调查核实要求，针对事实不清、要素不全的情况补充提供相应的事件信息，以形成完整清晰的事件终报。

# 3.2.4 事件信息终报

是指已完成事件初报或按需完成事件信息续报后，形成的要素完整、情况准确、格式规范、事实清晰的，符合公司安全网填报要求和局方安全信息系统上报规定的事件信息报告内容。

# 3.3 事发单位

是指与事件相关的飞行机组成员所属的、与所发生事件有关的、能提供事件直接信息的单位，通常是指人员所属飞行各部。

# 3.4 未按规定报告事件信息

# 3.4.1 迟报事件信息

是指责任机长个人或事发单位未及时上报事件信息，造成飞行部超过规定的事件信息上报时限，但在股份公司从局方或其他渠道得到相关信息前报告了信息，视为迟报事件信息。根据实际情形、原因和公司规定，分为个人迟报和单位迟报，分别落实管理。

# 3.4.2 隐瞒不报事件信息

是指责任机长个人或事发单位未及时上报事件信息，造成飞行部超过规定的事件信息上报时限，但股份公司已从局方或其他渠道得到

了相关信息，此时事发单位或个人无论上报与否，均视为隐瞒不报信息。根据实际情形、原因和公司规定，分为个人隐瞒不报和单位隐瞒不报，分别落实管理。

# 3.4.3 谎报事件信息

经调查，报告的信息内容与事实严重不符，视为谎报信息。包括责任机长个人或事发单位故意隐瞒或歪曲事实真相，隐藏、破坏或伪造证据，以及不配合续报及调查工作等情形。

# 3.5 航空器运行阶段

是指从任何人（飞行中应处于飞机上的人员）登上航空器准备飞行起至飞行结束这类人员离开航空器为止的过程。主要分为停放、推出/牵引、滑行、起飞、初始爬升、航路飞行（爬升/巡航/下降/等待）、进近、着陆、进位等阶段。

# 3.6 飞行中

是指自飞机为起飞而增加发动机功率达到起飞功率时起，至着陆滑跑减速至机型手册或本单位规定的最大直线滑行速度（二者取较大值）为止的过程。

注：飞行中包含中断起飞阶段。

# 3.7 飞行时间

航空器为准备起飞而依靠自身动力开始移动时起至飞行结束停止移动为止的时间。

# 3.8 机场活动区

是指机场内用于航空器起飞、着陆以及与此有关的地面活动区域，包括跑道、滑行道、机坪等。

# 3.9 飞行机组成员

飞行机组成员是指飞行期间在航空器驾驶舱内执行任务的驾驶员和飞行机械员。

# 3.10 航空器损伤

航空器损伤是指航空器（包括其部件和子系统）由于人为操纵或外部因素所导致的、且需要修复或修理的系统安全性或物理完整性缺陷。例如裂纹、断裂、变形、凹坑、刮痕、缺口、脱胶、分层、烧蚀、零部件缺失以及系统全部或部分失效等形式。

# 3.11 人员受伤

人员受伤的判定以最高人民法院、最高人民检察院、公安部、司法部颁发的《人体损伤程度鉴定标准》为准包括死亡、重伤、轻伤和轻微伤。

# 3.12 外来物

外来物是指机场活动区内无运行或航空功能并可能构成航空器运行危险的无生命的物体。

# 3.13 空飘物

空飘物是指在机场及周边区域飘浮的可能影响飞行安全的气球（不包括系留气球、无人驾驶自由气球和探空气球等）以及易被风吹起的边长或直径  $0.3 \mathrm{~m}$  以上的各类材质的块、条、幡、网、袋、膜等物体。

# 3.14 跑道侵入

在机场发生的任何航空器、车辆、人员错误地出现或存在于指定用于航空器着陆和起飞地面保护区的情况。

# 3.15 影响航空器安全运行

影响航空器安全运行的情况包括但不限于导致：航空器滑回、中断起飞、采取避让措施、空中等待、在最近合适机场着陆、改变进近方式、需转频联系、中止进近、复飞、返航、备降、跑道侵入、占用跑道/滑行道、实施跑道检查、需机场启动应急救援响应程序、紧急制动、航空器损伤等。

# 3.16 轻微伤

各种致伤因素所致的原发性损伤造成组织器官结构轻微损害或者轻微功能障碍。

# 3.17 轻伤

使人肢体或者容貌损害听觉、视觉或者其他器官功能部分障碍或者其他对于人身健康有中度伤害的损伤包括轻伤一级和轻伤二级。

轻微伤和轻伤的判定以最高人民法院、最高人民检察院、公安部、司法部颁发的《人体损伤程度鉴定标准》为准。

# 3.18 航空器着陆和起飞地面保护区

航空器着陆和起飞地面保护区是指跑道、位于适用的跑道等待位置和跑道之间的部分滑行道、跑道中线两侧各  $75\mathrm{m}$  范围内土面区、仪表着陆系统临界区/敏感区和跑道端安全区。

# 3.19 跑道侵入

在机场发生的任何航空器、车辆、人员错误地出现或存在于指定用于航空器着陆和起飞地面保护区的情况。

# 4 事件信息报告一般规定

# 4.1 事件信息报告原则

涉及事件的责任机长和事发单位应按照“快报事实、慎报原因、实事求是、依法处置”的总体原则及时、准确、如实报告事件信息，不得隐瞒不报、谎报或者迟报。

# 4.2 事件信息报告时限

4.2.1 对于紧急事件，在确保飞行安全的前提下，责任机长应第一时间报告，不得晚于事发后30分钟；对于所有敏感事件，通常情况下在航班到达停机位完成关车程序后，即具备首报条件，责任机长或事发单位应在关车后30分钟内，向飞行部安全管理部通过电话完成首报。  
4.2.2 所有事件信息，都应在航班运行结束后尽快提交初报；紧急事件，应在航班运行结束后立即提交机长报告，紧急事件初报提交后，事发单位必须通知安管部信息员，告知初报情况；非紧急事件应在航班运行结束后尽快提交初报，非紧急事件提交初报时间不得晚于关车时间后12小时；  
4.2.3 局方规章和公司规定中明确的诸如“X小时内或XX小时内按规范如实在公司安全网上填报信息”“X小时内或XX小时内按

规范如实填报民用航空安全信息报告表，主报事发地民航监管局，抄报事发地民航地区管理局、所属地民航监管局及民航地区管理局”等规定，适用于飞行部对接公司或公司对接局方的安全信息管理规定，用以从法规层面落实对运行单位和公司的安全信息主体责任和管理奖惩，不适用于对于责任机长个人或事发单位的事件信息报告时限。

# 4.3 事件信息报告途径

4.3.1 事件信息首报途径为电话联系安全管理部24小时信息值班人员，24小时值班人员电话见本规定首页。若出现电话未接通情况，请机组及时联系事发单位管理干部，由其告知安管部相关人员。  
4.3.2 事件信息初报、续报途径为责任机长通过 EFB 电子任务书记录报告，或事发单位通过公司三期安全网，填报事件报告内容并提交。各机队信息员在要求时限内核实、处理或提交报告。  
4.3.3 事件信息终报，由飞行部安全管理部通过公司安全网提交，或按照公司要求，在局方安全信息系统填报。

# 4.4 事件信息报告责任

涉及事件的责任机长对事件信息的报告负责，事发单位对本单位机长报告制度的落实负主体责任，对事件信息报告的及时性、有效性、完整性、规范性、真实性负责。

# 4.5 事件信息报告保护

责任机长和事发单位在信息报告过程中，应遵守《民用航空安全信息保护管理办法》、公司《安全管理手册》安全信息保护的相关规定和《上海飞行部民用航空安全信息保护管理细则》，规范事件信息

的上报、收集、应用流程，强化管控办法，避免出现不当披露和公开的情况，未经飞行部批准，任何个人和单位不得对外发布。

# 4.6 其他/使用原则

4.6.1 “事件信息”初报标题：日期（航班计划日期）+航空器使用单位+机型+机号+航班号+起飞机场+计划落地机场+事件类型+事件后果（如可确认）。

注：标题内容在事件类型后加入相应的事件后果描述；

对于不同的事件需要对应不同的事件后果描述。（例如：遭遇鸟击无损伤/有损伤/损伤情况未知等）；

4.6.2 “事件信息”初报中的“日期”应填写为航班计划日期（北京时），此日期可能与事件发生日期不一致；事件信息中的“日期”与“时间”均为北京时（国际航班应特别注意时差）；  
4.6.3 “事件信息”初报内容中，关于“告警信息”和“机械故障”的描述应以中文为主，并备注英文。例如：“发动机1失效（ENG1FAIL）”  
4.6.4 旅客吸烟引起的烟雾告警事件，在信息初报中应明确是“烟草”或“电子烟”；  
4.6.5 假告警的事件，应当在事件报告内容中描述的告警后注明“假告警”。例如：触发 PULL UP 告警（假告警）；  
4.6.6 涉及鸟击相关事件，事件信息报告中必须明确鸟击高度或根据目视鸟（群）、听到异响、观察到飞机参数异常/其他状态异常等情况判断的疑似鸟击高度；

4.6.7 事件信息报告中涉及油量情况时，应填写落地剩余油量和机型最后储备油量。  
4.6.8 对于飞行时间内因吸烟、浮尘、空气清新剂、杀虫剂、发胶等气溶胶制品引起的烟雾假警告，地形数据库不正确、GPS 信号丢失引起的近地假警告事件无需通过电话向飞行部安全管理部首报，但应当在航班运行结束后立即按紧急事件要求提交机长报告；

# 5 事件信息报告流程

# 5.1 流程概述

5.1.1 事件信息报告流程主要包括首报、初报、续报、终报四个环节，总体要求是：

a) 首报要快（事发后半小时、不得晚于关车后半小时）  
b) 初报要实（及时按规定格式和要素报告现象、经过和结果）  
c) 续报要准（完善缺失的内容，回复需核实的情况）  
d) 终报要全（格式规范、要素完整、事实清晰）

5.1.2 事件信息总体包括民用航空器事故、民用航空器征候以及民用航空器一般事件信息。根据事件实际情况和上报规定，将事件信息报送流程区分为涉及“敏感事件”的流程，和不涉及“敏感事件”的流程两大类，敏感事件定义和范畴见3.1.4“敏感事件”。

a) 涉及“敏感事件”的事件信息报告流程为首报、初报、续报、终报；  
b) 不涉及“敏感事件”的事件信息报告流程为初报、续报、终报；

5.1.3 初报信息完整、清晰、规范，符合事件信息报告终报规定，则无需进行续报，可直接作为终报完成事件信息报告。

# 5.2 事件信息首报

5.2.1 事件发生后，责任机长可通过本规定附件内容对照是否涉及敏感事件，包括民用航空器事故、民用航空器征候、紧急事件、公司严重差错、触碰安全红线事件，并结合运行中受到局方调查问询、外单位质询或上报实际情况，按规定落实电话首报要求。  
5.2.2 敏感事件发生后，由责任机长或事发单位，在可能的情况下（事发后半小时内）通过电话联系安全管理部信息值班人员，本着“快报事实、慎报原因”的原则，向上海飞行部安全管理部报告敏感事件的事实和现象等信息。  
5.2.3 敏感事件信息首报内容应当完整、准确、规范，基本要素包括但不限于：事件的发生时间、地点、单位、机型机号、航班号、简要经过、事件现象、处置情况、航空器损伤、人员受伤和事件影响等。  
5.2.4 敏感事件电话首报报告基本模板：XXXX 年 X 月 XX 日（航班计划日期），上海飞行部 X 部 X 分部，机型 XXXX-XXXX/B-XXXX号机执行 MUXXXX（XXXX-XXXX）航班，机组成员：左座 PFXXX（派遣资质），右座 PM XXX（派遣资质），观察员座：XXX（派遣资质）。北京时 XX:XX（事发时间），在 XXXX 区域 XXXX 阶段，出现/遭遇XXXXXXX情况，机组处置情况，处置的结果，人员情况（飞行机组、客舱机组、客舱人员等），飞机状态（是否超出飞行包线、是否出现设备损伤、系统工作情况、应急设备使用情况等）。

# 5.3 事件信息初报

5.3.1 事件发生后，责任机长可根据本规定 3.1.1 事件信息定义，判断是否需要报告事件信息。按照公司运行手册规定，“执行航班任务中任何偏离正常运行的情况”都应提交事件信息报告，确保应报尽报。  
5.3.2 事件发生后，责任机长应在航班运行结束后，尽快通过 EFB 提交事件信息初报；或报请所在单位核实事件情况后，由所在单位信息员通过公司安全网向飞行部安全管理部提交事件信息初报；对于责任机长无法及时通过 EFB 提交事件信息初报的情况，责任机长必须通知所在单位，要求所在单位及时通过公司安全网向飞行部安全管理部提交事件信息初报。

5.3.3 事件信息初报应确保信息的完整性、规范性，需包含以下基本内容：

a) 日期、所在单位、机型、机号、所执行航班、人员在座情况；  
b) 发生地：地点、区域、飞行阶段；  
c) 事件经过：简要经过中，应当根据实际情况如实、完整描述事件相关的现象、过程和结果，内容真实确切，故障信息使用中文描述、英文备注；  
d) 人员、设备、天气状况;  
e) 与事件有关的其他信息;

5.3.4 事件信息初报基本模板：XXXX年X月XX日（航班计划日期），上海飞行部X部X分部，机型XXXX-XXXX/B-XXXX号机执行

MUXXXX（XXXX-XXXX）航班，起飞时间XX：XX，着陆时间XX：

XX，机组成员：左座 PFXXX（派遣资质），右座 PM XXX（派遣资质），观察员座：XXX（派遣资质）。北京时 XX:XX，在XXXX区域XXXX阶段，北京时间XX：XX，出现/遭遇/触发XXXXXXXX情况，机组处置情况，处置的结果（非正常/应急情况/告警的现象和持续时间等），对运行造成的影响情况（改变高度/航迹/程序、复飞、返航、备降、滑回等），人员情况（飞行机组、客舱机组、客舱人员等），飞机状态（是否超出飞行包线、是否出现设备损伤、系统工作情况、应急设备使用情况等），后续运行情况。

5.3.5 事件信息初报可参考本规定附件《典型事件信息填报示例》填报，未包含于示例中的情况，需按照本规定 4.3.3 和 4.3.4 明确的基本规定填报。  
5.3.6 涉及事件的责任机长和事发单位，需检查并确保所填报的事件信息初报内容，要素完整、情况准确、格式规范、事实清晰、故障信息使用中文描述英文备注，并正确上传至公司安全网。

# 5.4 事件信息续报

5.4.1 责任机长或事发单位完成事件信息初报的填报后，对事件信息存在补充需求，或事发单位获得新的事件情况时，应主动补充填报并更新所提交的事件信息初报，执行事件系信息续报。  
5.4.2 责任机长或事发单位完成事件信息初报的填报后，经飞行部安全管理部审核，若事件信息的完整性、规范性不满足事件信息报告的基本要求，应按照飞行部安全管理部要求，及时通过重新填报或补充填报的方式执行事件信息续报。

5.4.3 责任机长或事发单位应及时按照局方、公司以及飞行部安全管理部对事件信息的调查核实要求，针对事实不清、要素不全的情况补充提供相应的事件信息，包括事件有关的文本、影音、数据以及其他资料，以形成完整清晰的事件报告。

# 5.5 事件信息终报

责任机长或事发单位提交的事件信息初报，或按需提交的事件信息续内容，要素完整、情况准确、格式规范、事实清晰，符合公司安全网上报要求和局方安全信息系统填报规范，并已及时、正确上传至公司安全网，即可作为事件信息终报完成事件信息报告。

# 6 事件信息报告管理办法

# 6.1 违规情形一

对于责任机长违反事件信息报告规定，但未造成飞行部违反公司规定和局方法规的情况，根据飞行部《违章无后果事件管理规定》一般违章无后果事件条款，对责任机长个人启动违章无后果事件调查并根据情形进行相应处理；对事发单位未落实本单位机长报告制度的管理责任，连续发生此类事件的情况，对责任机长所在单位按照飞行部SMS评审标准进行考核。

# 6.2 违规情形二

对于责任机长违反事件信息报告规定，或事发单位未落实本单位机长报告制度的管理责任，出现隐瞒不报、谎报、迟报安全信息的情况，造成责任机长个人或飞行部受到公司处理或考核的情况，在公司

处理和考核的基础上，根据事件情形对责任机长个人参照一般差错相应标准进行处理，对事发单位参照一般差错相应标准进行考核。

# 6.3 违规情形三

对于责任机长违反事件信息报告规定，或事发单位未落实本单位机长报告制度的管理责任，出现隐瞒不报、谎报、迟报安全信息的情况，造成责任机长个人、飞行部或公司被局方追究法律责任，处以行政处罚的情况，在责任机长和事发单位承担相关责任的基础上，根据事件情形对责任机长个人参照严重差错相应标准进行处理，对事发单位参照严重差错相应标准进行考核。

# 6.4 违反安全信息保护规定的情形

情节轻微未造成后果的情况，对责任人员或单位进行批评教育；情节严重或造成敏感安全信息泄露的，依规依纪依法给予处分。

# 7 附则

7.1 本规定未明确的规定及事宜，以《中国东方航空集团有限公司安全管理手册》《东方航空股份有限公司安全管理程序和规定》《中国东方航空集团有限公司安全信息和事件调查管理办法》的规定、程序和办法为准。  
7.2 本规定未尽事宜，遵照飞行部安委会决议执行。

# 附件一

# 事件样例

# 一、运输航空紧急事件样例

1.航空器空中相撞、坠毁或迫降。  
2. 飞行中, 航空器失控、失速, 出现失速警告、抖杆累计 3s (含)以上。  
3. 飞行中, 挂碰障碍物 (含升空物体) 或起落架机轮 (滑橇、尾环、浮筒) 之外的任何部位触地/水。  
4. 低于安全高度需机组立即采取措施或触发拉起（Pull-Up）地形警告。  
5. 冲/偏出跑道、滑行道或跑道外接地。

注：含中断起飞停在停止道的情形。

6. 在航空器起飞阶段或进近着陆阶段机场标高  $60 \mathrm{~m}$  以下发生的跑道侵入。  
7. 在滑行道，或未指定、关闭、占用的跑道（不含跑道侵入）上，起飞、中断起飞、着陆或从机场标高  $300 \mathrm{~m}$  以下复飞。

注 a: 由于超出空中交通管制 (ATC) 正常预期占用跑道 (例如: 航空器突发机械故障、脱离动作慢、起飞离地晚 、机场实施跑道检查) 导致航空器中止进近或复飞的情形除外。

注 b：包括复飞过程中低于机场标高  $300 \mathrm{~m}$  的情况。

8. 飞行中，飞行机组成员因受伤、患病、疲劳、酒精、食物中毒或药物的影响而无法履行其职责。  
9. 飞行中，出现座舱高度警告、座舱高度达到该运行阶段应当触发座舱高度警告的条件或客舱氧气面罩自动脱落的情况，以及出现烟雾或毒气等需要飞行机组成员使用氧气的紧急情况。  
10. 航空器起火、冒烟；发动机起火，或出现火警；飞行时间内，航空器出现火警、烟雾警告。

注：因刹车引起的轮毂冒烟和烤箱内食物冒烟以及未引起烟雾告警的旅客吸烟事件除外。

11.非包容性涡轮发动机失效；飞行时间内，出现任意一台发动机停车或需要关停的情况。  
12. 飞行中，导致航空器操纵困难（以飞行员判断为准）的系统故障、部件脱落、天气现象、飞行超出批准的飞行包线或其他情况；接到飞行员报告“操纵困难”。  
13.低于运行标准（机场运行最低标准、航空器运行标准、飞行机组资格标准）起飞、开始最后进近或着陆。  
14. 未取下操纵面夹板、起落架安全销、挂钩、空速管套、静压孔塞或尾撑杆起飞。  
15. 需要机组成员宣布遇险状态（Mayday）、宣布紧急状态（PanPan）、设置应答机编码 7700 或需要紧急撤离的情况。

注：因天气绕飞、旅客患病宣布紧急状态的情况除外。

16. 飞行中，航空器与航空器之间小于规定间隔（不考虑容差）或平行跑道同时仪表进近运行时航空器进入非侵入区（NTZ）。  
17. 飞错航路、飞偏或飞错进离场航线、未正确执行复飞程序并导致其他航空器避让（例如：调整高度、调整航向、调整航路）。  
18. 迷航，误入禁区、危险区、限制区、炮射区，误入或误出国境。  
19. 区域范围内，陆空通信双向联系中断 20min（含）以上；飞行中，进近或塔台范围内，陆空通信双向联系中断 3min（含）以上（通航使用机载设备以外的方式建立可靠通信联系的情况除外）。

注：陆空通信双向联系中断时间的确定为自飞行机组或空管单位发现通信联系中断时起，至重新建立通信联系时止。  
20. 航空器与航空器碰撞；或航空器与设施设备、车辆、人员或其他物体碰撞，导致航空器受损。  
21.因航空器原因需机场启动紧急出动等级的应急救援响应。  
22. 人员死亡、重伤，或航空器运行、维修或保障过程中，导致人员轻伤。

注：人员患病引起的受伤情况除外。

# 二、运输航空非紧急事件样例

# 航空器运行

1. 飞行机组成员在飞行中以外的运行阶段以及客舱乘务员和随机机务人员在运行阶段因受伤、患病、疲劳、酒精、食物中毒或药物的影响而无法履行其职责。  
2. 航空器未按规定进行除/防冰，航空器起飞。  
3. 不符合放行条件放行或未执行放行工作，航空器起飞。  
4. 未得到管制许可推出、起动、滑行、起飞或着陆。  
5. 发生跑道侵入。  
6. 航空器滑错滑行道需要使用飞机牵引设备重新回到正确滑行路线，或直升机在指定的起降坪（平台）外接地。  
7. 航空器与设施设备、车辆、人员、动物或其他地面障碍物相碰撞；航空器与航空器、设施设备、车辆、人员、动物或其他地面障碍物存在碰撞可能，导致航空器紧急制动、改变方向；航空器非正常位移。  
8. 在滑行或飞行中，燃油、滑油或液压油渗漏（按手册未超标的情况除外）。  
9. 航空器动力装置所产生气流导致航空器损伤、地面设施设备损伤或人员受伤。  
10. 航空器遭外来物撞击，导致航空器损伤。

注：外来物损伤航空器的确认参照《运输机场外来物防范管理办法》

11.航空器轮胎爆破、脱层或扎破处遗留外来物。  
12. 航空器携带其他物体飞行。

注：其他物体包含遗留在电子舱、起落架舱、盖板、雷达罩、发动机、APU、油箱、操纵面、货舱（含有锂电池的物体和未按规定申报而运输的危险品）的物体。  
13. 由于乘员、行李、邮件、货物、压舱物等的重量、装载舱位与固定等原因，导致超过最后一分钟修正限值或超出审定重心限制，航空器起飞。  
14. 载重平衡舱单飞机基础数据错误或计算/输入与实际不符，航空器起飞。  
15. 载运的物品因泄漏、位移等情况导致航空器损伤或人员受伤。  
16. 滑梯包掉落、滑梯/救生筏放出或应急出口非正常打开。  
17.机上人员非正常触发机载应急定位发射机（ELT）。  
18. 航空器中断起飞。  
19. 飞行中，未完成预定的航空器构型。  
20. 飞行中, 出现失速警告、抖杆或自动保护 (如 Alpha Floor)。  
21. 航空器俯仰角超过  $+25^{\circ}$  或  $-10^{\circ}$  、坡度超过  $45^{\circ}$  。

22. 超过飞机飞行手册（AFM）/飞行机组操作手册（FCOM）/飞机维护手册（AMM）限制数据（上述手册中最严标准），如：过载（G值）、重量、发动机排气温度、振动值、速度、客舱压差或轮速等。

其中，对于无超速警告的机型，“超过襟翼标牌速度”“超过最大操纵空速”“超过最大马赫数”持续2秒及以上。

注：着陆垂直过载以手册和下列标准的较低值为准：窄体机 2.2（含）、宽体机 2.0（含）。侧向过载超限除外。

23.航空器系统失效/故障或零部件缺失，导致中断起飞、采取避让措施、快速下降、改变进近方式、中止进近、复飞、返航、备降、占用跑道、需机场启动应急救援响应程序或需在最近合适机场着陆；飞行中，航空器驾驶舱窗出现裂纹。  
24. 飞行中，航空器增压异常导致氧气面罩脱落或需改变高度；运行阶段，人为原因导致氧气面罩脱落。  
25. 飞行中遇有颠簸或其他原因导致航空器损伤或人员受伤。  
26. 飞行中，航空器遭遇风切变或触发风切变警告警戒，需机组采取措施。  
27.50 英尺（无线电高度）以下的复飞；1000 英尺（无线电高度）以下的人为原因复飞。  
注：包括复飞过程中低于无线电高度50英尺的情况。  
28. 除低能见度、大风、乱流、雷雨等天气原因，机场关闭、宵禁和旅客原因外，返航、备降。  
29. 航空器遭雷击、电击、冰击、雹击、其他物体（不含外来物）撞击，导致航空器损伤。  
30. 由于冰、雪、霜、雨、沙尘或火山灰等在航空器表面或动力装置积累，导致航空器操纵特性降低和性能明显下降。

31. 航空器遭鸟击（含蝙蝠），留下血迹、羽毛、皮肤、肌肉或肢体等残留物，且导致航空器损伤。

注 a: 机场内鸟击是指航空器起飞和初始爬升阶段高度机场标高  $100 \mathrm{~m}$  (含) 以下以及进近着陆阶段高度机场标高  $60 \mathrm{~m}$  (含) 以下发生的鸟击。

# 一、运输航空严重征候

1. 为避免航空器相撞或其他不安全情况，应做出规避动作的危险接近。  
注：在程序管制区域，垂直间隔和水平间隔同时小于 1/5 规定间隔；在雷达 / ADS-B 管制区域，垂直间隔和水平间隔同时小于规定间隔，且危险指数大于 90（含）的飞行冲突。  
2. 飞行中，未被定性为事故的相撞。  
3. 几近发生的可控飞行撞地。  
4. 在滑行道，或未指定、关闭、占用的跑道上中断起飞。  
5. 在滑行道，或未指定、关闭、占用的跑道上起飞。  
6. 在滑行道，或未指定、关闭、占用的跑道上着陆或尝试着陆。

注：由于前机超出空中交通管制（ATC）正常预期占用跑道（例如：突发机械故障、脱离动作慢、起飞离地晚）导致后机中断着陆或进近的情形除外。

7. 未被列为事故的任一起落架收回着陆。  
8. 飞行中，航空器机轮之外的其它部位擦地。

注：以下情况除外：

a）未导致航空器受损的机尾（不含尾橇）擦地；  
b）仅擦尾橇且未导致除尾橇之外的航空器其它部位受损。

9.在起飞或初始爬升过程中明显未达到预定性能。

10. 飞行中，驾驶舱、客舱和货舱起火或冒烟，或发动机起火，即使这些火被扑灭。

注：以下情况除外：

a）机上人员携带的电子设备（包括移动电源、移动通信设备、平板电脑、摄录设备等）的锂电池起火或冒烟，机组成员及时发现并妥善处置，且未导致航空器受损和／或人员轻伤的情况；  
b）灯泡冒烟，烤箱中食物或食物残留导致冒烟，液体滴溅到操纵台面板上导致冒烟，引气或空调管路中残留物产生烟雾的情况。  
11. 飞行中，需要机组人员紧急使用氧气的情况。  
注：座舱高度达到客舱氧气面罩自动脱落触发条件的情况（货机参考同类或同类可比客机）  
12. 未被列为事故的航空器结构受损或发动机解体，包括非包容性涡轮发动机失效。  
13. 飞行中，严重影响航空器运行的一个或多个系统出现的多重故障。  
14. 飞行中，飞行机组成员丧失工作能力，符合下列情形之一的：

a）导致飞行机组成员数量或资质不满足该机型的最低配置；  
b）在飞行关键阶段，飞行机组成员在飞行操作岗位丧失工作能力。  
15. 燃油量或燃油分布需要飞行员宣布紧急状态的情况。  
16.A 类跑道侵入。

17. 起飞或着陆中，冲出、偏出跑道或跑道外接地。  
18. 导致航空器操纵困难的系统故障、天气现象、飞行超出批准的飞行包线或其他情况。  
19. 飞行中，必需的飞行引导与导航冗余系统中一个以上的系统失效。  
20. 飞行中遇有颠簸或机组操纵等原因导致 3（含）人以上人员轻伤。  
21. 类似上述条款的其他事件。

# 二、运输航空一般征候

1. 为避免航空器相撞或其他不安全情况，应做出规避动作的危险接近。

注：在程序管制区域，垂直间隔和水平间隔同时小于1/3但未同时小于1/5规定间隔；在雷达/ADS-B管制区域，垂直间隔和水平间隔同时小于规定间隔，且危险指数介于75（含）至90之间。尾流间隔小于1/2规定间隔。

2. 有发生可控飞行撞地风险。

注：危险指数介于 75（含）至 90 之间的。

3. 在滑行道，或未指定、关闭、占用的跑道上，仪表进近时从机场标高  $300 \mathrm{~m}$  至决断高度（高）或最低下降高度（高）复飞；目视进近时从机场标高  $150 \mathrm{~m}$  至机场标高  $60 \mathrm{~m}$  复飞。

注：由于前机超出空中交通管制(ATC)正常预期占用跑道（例如：突发机械故障、脱离动作慢、起飞离地晚）导致后机中断着陆或进近的情形除外。

4. 在起飞、着陆或复飞过程中，在跑道上擦机尾，未导致航空器受损，或仅需维修/更换尾橇。  
5. 除飞行中以外的运行阶段，驾驶舱、客舱和货舱起火或冒烟，或发动机起火，即使这些火被扑灭。

注：以下情况除外：

a）机上人员携带的电子设备（包括移动电源、移动通信设备、平板电脑、摄录设备等）的锂电池起火或冒烟，机组成员及时发现并妥善处置，且未导致航空器受损和/或人员轻伤的情况；  
b) 灯泡冒烟, 烤箱中食物或食物残留导致冒烟, 液体滴溅到操纵台面板上导致冒烟, 引气或空调管路中残留物产生烟雾的情况。  
6. 飞行中出现任意一台发动机停车/失效或需要关车的情况。  
7. 飞行中，除“严重征候14”之外，飞行机组成员丧失工作能力，符合下列情形之一的：  
a）导致其他飞行机组成员的飞行时间超过《大型飞机公共航空运输承运人运行合格审定规则》（CCAR-121）规定的时限；  
b）飞行关键阶段以外在飞行操作岗位丧失工作能力。  
8. B类跑道侵入。  
9. 在着陆过程中，航空器起落架未放到位，且高度下降到机场标高  $100 \mathrm{~m}$ （含）以下。

10. 飞行中遇有颠簸或机组操纵等原因导致 3 人以下人员轻伤。  
11.平行跑道同时仪表运行时，机组没有正确执行离场或者复飞程序导致其他航空器避让，或者管制员错误的离场或复飞指令导致其他航空器避让。  
12. 平行跑道同时仪表运行时，航空器进入非侵入区(NTZ)，导致其他航空器避让。  
13. 航空器未按性能计算结果而设定的起飞构型继续起飞。  
14. 区域范围内陆空通信双向联系中断  $30 \mathrm{~min}$  (含) 以上; 或双向联系中断  $20 \mathrm{~min}$  (含) 以上, 且导致航空器小于规定间隔。进近或者塔台范围内陆空通信双向联系中断  $3 \mathrm{~min}$  (含) 以上, 且导致航空器小于规定间隔。  
15. 误入禁区、危险区、限制区、炮射区或误出、入国境。  
16.迷航。  
17. 未取下操纵面夹板、挂钩、空速管套、静压孔塞或尾撑杆等而起飞。  
18. 飞偏或飞错进离场航线并导致其他航空器避让。  
19. 航空器部件缺失、蒙皮揭起，且导致航空器受损。  
20.轮胎爆破或脱层，导致航空器其他部位受损。  
21. 飞行中遭雷击、电击、鸟击、冰击、雹击、外来物或其他物体撞击，导致航空器受损。  
22.在飞行中以外的运行阶段，航空器与航空器、设施设备、车辆、人员或其他地面障碍物相撞，导致航空器受损或人员轻伤。

23. 由于航空器内货物、邮件、行李、集装器等的装载与固定等原因，导致航空器受损，或飞行中超出重心限制。  
注：仅导致航空器货舱地板、壁板受损情况除外。  
24.航空器载重平衡计算或输入与实际不符，导致飞行中超出重心限制。  
25. 危险品破损、溢出、渗漏或包装未能保持完整等情况，导致航空器受损或人员轻伤。  
26. 餐车、储物柜等客舱内设施设备滑出或跌落，导致航空器受损或人员轻伤。  
27. 飞行中由于过载，导致航空器受损。  
28. 航空器超过最大允许起飞重量起飞。航空器超过最大允许着陆重量着陆并导致航空器受损。  
29. 飞行中出现失速警告持续 3s（含）以上（假信号除外）。  
30. 航空器携带其他物体飞行，导致航空器受损或影响操纵。  
31. 类似上述条款的其他事件。

# 三、民用航空器征候样例清单

# (一) 运输航空严重征候样例

样例 1. 与无人机等相撞。  
样例 2. 危险指数大于 90（含）的。  
样例 3. 飞行中挂碰障碍物。

样例4. 仪表进近认错跑道（包括跑道方向）进近，且在决断高度（高）或最低下降高度（高）以下复飞。

样例5. 目视进近认错跑道（包括跑道方向）进近，且在机场标高60米以下复飞。

样例6. 飞行中，起落架舱、起落架、电子舱、辅助动力装置起火。

样例 7. 飞行中, 机载电子、电气设备起火或冒烟。

样例8. 航空器仅靠备用电源飞行，或仅靠备用和应急液压系统飞行。

样例9. 飞行中多于一台发动机同时停车。

样例 10. 航空器着陆接地时油量少于紧急油量。

样例11. 飞行中航空器失速。

样例 12. 飞行中任一主操纵系统完全失效。

样例 13. 在不满足条件的跑道上起飞、中断起飞或着陆。

样例 14. 航空器迫降。

样例 15. 飞行中进入积雨云、浓积云，导致航空器受损。

样例 16. 航空器不能保持安全高度。

样例 17. 低于最低运行标准起飞或着陆。

样例18. 航空器未按规定完成除、防污染作业（含冰、雪、霜、虫等）起飞。

样例 19. 飞行中反推意外打开。

# （二）运输航空一般征候样例

样例1. 航空器进入俯仰角超过  $+35^{\circ}$  或  $-15^{\circ}$ , 或坡度超过  $60^{\circ}$ ,或空速与当时飞行状态不相匹配的情况。

样例2. 飞行中，发生非指令性安定面配平。

样例3. 在起飞滑跑或最后进近阶段，航空器遭遇风切变或触发风切变警告，未按要求中断起飞或复飞的情况。

# 附件三

# 中国东方航空股份有限公司需关注类事件判定标准

# 一、目的

为加强公司航空安全管理工作，规范安全从业人员的操作行为，公司将持续关注在组织和实施生产运行及相关的生产活动中管理原因或人为原因的航空不安全事件，通过及时采取防范措施，前移安全关口，控制风险，消除隐患，预防事件的发生。

# 二、依据

民航局《事件样例》AC-396-08R3

民航局《运输航空公司一般事件判定指南》MD-396-03

公司《安全管理手册》

# 三、适用范围

本标准适用于中国东方航空股份有限公司范围内发生，由于管理或人为责任原因影响或可能影响航空安全的事件，其性质和程度尚未构成民用航空器征候、一般事件（一类事件和二类事件）。本标准请上航、云南公司、东航物流和中联航参照执行。

# 四、术语定义

# （一）航空安全差错

在组织实施生产运行及与之相关的生产活动中，由于管理或人为责任原因危及航空安全，其性质和程度尚未构成民用航空器征候或相应等级，但影响或可能影响安全的事件。根据不安全事件的性质及

对安全的影响程度，航空安全差错划分为严重差错和一般差错两个等级。

# （二）航空器运行阶段

从任何人登上航空器准备飞行起，至飞行结束这类人员离开航空器为止的过程。

# （三）飞行中

自航空器为实际起飞而使用动力时起，至着陆冲程终止的过程（包含中断起飞阶段）。

# （四）机场活动区

机场内用于航空器起飞、着陆以及与此有关的地面活动区域，包括跑道、滑行道、机坪等。

# （五）机场控制区

是指根据安全需要在机场内划定的进出受到限制的区域。

# （六）航空器损伤

航空器（包括其部件和子系统）由于人为操作或者外部因素所导致的、且需要修复或者修理的系统安全性或物理完整性缺陷。例如裂纹、断裂、变形、凹坑、刮痕、缺口、脱胶、分层、烧蚀、零部件缺失以及系统全部或者部分失效等形式。本差错标准的损伤为直接经济损失金额在10万元（含）以上。

# （七）航空器受损

航空器受损程度低于航空器放行标准，仅轮胎损坏，或使用打磨、填充、粘贴金属胶带、补漆、冲洗、安装临时紧固件等方式进行临时修理后符合放行标准的情况除外。

# （八）直接经济损失

航空器、车辆、设备及设施的修复费用，包括器材费、工时费、运输费等，以人民币计算。

# （九）最大业载

本次航班所允许业载的最大值。

# （十）危险品事件

不同于危险品事故，但与危险品航空运输有关联，不一定发生在航空器上，但造成人员受伤、财产损失、起火、破损、溢出、液体或放射性物质渗漏或包装未能保持完好的其它情况。任何与危险品航空运输有关并严重危及航空器或机上人员的事件也视为危险品事件。

# （十一）外来物

机场活动区内无运行或航空功能并可能构成航空器运行危险的无生命的物体。

（十二）车辆、地面设施设备、客舱设施设备受损指直接经济损失金额在5万元（含）以上的受损。

# （十三）人为原因

如果操作人员的行为是导致事件发生的直接原因或主要原因，那么事件发生的原因就被认定为人为原因。人为原因既包括违规违章，也包括失误疏忽。

# （十四）管理原因

在事件调查中发现管理者或管理层职责不明确、规章制度不健全、无相应管理机构、无有效工作程序等情况。

# （十五）轻微伤

各种致伤因素所致的原发性损伤，造成组织器官结构轻微损害或者轻微功能障碍。人体损伤的具体界定，详见最高人民法院、最高人民检察院、公安部、司法部2013年8月30日颁发自2014年1月1日起施行的《人体损伤程度鉴定标准》。

（十六）重心偏差是指起飞重心、着陆重心、无油重心偏差中取最高值。  
（十七）一个不安全事件同时符合公司数个具有适用关系的具体条款时，依规就高处理。

# 五、关注类事件

在飞机运行、维修或保障过程中，发生下列未构成事件等级，可能导致飞机损伤、人员受伤或其他影响飞行安全等后果的情况，飞行各部应按要求开展信息报告、事件调查，并将相关工作完成闭环。

4.1 飞机与飞机、设施、设备、货物、车辆等物体相刮碰。  
4.2 载运、储存、停放的物品冒烟、起火、爆炸；在飞机上或装/卸机过程中危险品泄漏。  
4.3人为原因导致机上设备损坏或失效，影响安全。

4.4 由于操作错误或维护不当，导致为飞机提供服务的车辆、设施设备等起火。  
4.5 未及时处理事件信息、运行限制信息、航行通告等，造成超限制运行、进入不安全区域等后果。  
4.6 机组成员、飞行签派员、机务人员违反公司《运行手册》规定的值勤、飞行、休息时间限制（执行特殊任务除外）。  
4.7 飞机运行、维修或保障过程中，造成人员受伤。  
4.8 生产运行和安全管理人员不满足资质要求。  
4.9 驾驶舱侧窗、客舱门、货舱门夹带异物或未关好，且航空器起飞。  
4.10 飞机误入未启用的跑道，滑/拖错滑行道、停机位或滑/拖过等待线，偏离机型标示停止线或中心滑行线需进行调整。  
4.11 未按程序或检查单操作，遗漏或错误执行关键操作/重要动作，未构成二类事件。  
4.12 未按规定操作相关运行系统，造成关键生产系统不能正常工作，影响运行。  
4.13 未按规定对获得的关键信息进行正确处置，影响航空器安全运行。  
4.14 民航局关注或造成舆情类事件。  
4.15 未执行飞行程序，导致起飞时出现起飞形态警告。

4.16 机组与管制通信双向联系中断。  
4.17 飞机偏离指定航线（迹）/航路，进、离场程序，或高度（颠簸导致高度偏离除外）。  
4.18 起飞或复飞时出现 DON'TSINK 告警；进近时出现 SINKRATE、GLIDESLOPE 等告警持续 5 秒（含）以上。  
4.19 仪表进近机场标高 1000 英尺以下未建立稳定进近; 目视进近机场标高 500 英尺以下未建立稳定进近。  
4.20 飞行中出现 BANKANGLE 告警或坡度达到 35 度（含）以上；未按规定完成航空器除/防冰作业航空器滑出；人为原因导致机场标高 1000 英尺以下的复飞。  
4.21 启用反推后，未实施全停，继续起飞或复飞。  
4.22 人为原因导致飞机中断起飞。  
4.23 指示空速超出机型规定的飞机形态限制速度持续 4 秒（含）以上，且峰值大于限制 5 节（含）。  
4.24 机组操纵不当触发 A-FLOOR。  
4.25 错误调置进近导航台、航道，继续进近。  
4.26 停场维修期间，飞机滑梯、救生筏非正常释放，或滑梯包掉落。  
4.27 放行错误，或未按照手册、程序要求放行飞机。  
4.28 未按指令或规定起动发动机、试车。  
4.29 飞机的各种盖板（含滑油箱加油口盖）未关好或堵塞、锁销等未取下，飞机依靠自身动力滑行。

4.30 飞机发动机、起落架、液压、燃油、飞行操纵等系统维护过程中漏检、漏装或错装部件，且飞机已放行。  
4.31 未按规定取下与飞机连接的电源线、空调管、气源管、加水管等相关设施设备，造成飞机插头、接口或机体损伤。  
4.32 由于人员错误操作，造成飞机轮胎易熔塞熔化。  
4.33 运行阶段，由于未按规定操作，导致飞机部件脱落。  
4.34 由于人员错误操作，导致发动机地面非正常停车。  
4.35 飞机停场维修期间，未按规定操作导致人员受伤、飞机受损或设备损坏。  
4.36 客舱内设备、行李、其他物品滑出或其他原因（旅客自身原因除外）造成机上设备损坏。  
4.37 飞行关键阶段在客舱通道、出口及没有固定装置的隔间堆放行李物品。  
4.38 未按规定固定餐车、储物柜等客舱设备，导致直接经济损失超过1万元（含）。  
4.39 在直接准备阶段，未按规定检查客舱应急器材、设备，造成影响。  
4.40 货物、行李、邮件发生破损、外溢、泄漏污染飞机，需机务做专业处理。  
4.41 载运的行李、货物或集装器航班运输过程中发生坍塌、位移。

4.42 货物（含危险品）违规运载。  
4.43 未按规定运输活体动物, 造成活体动物逃出外包装。  
4.44 在机场控制区内，未经授权操作车辆或设备。  
4.45 未按规定操作危险品，包括运输未申报或错误申报的危险品，运输未按规定进行包装的危险品，运输未按规定向机长提供信息的危险品，未规定组装、存储、装卸、隔离、分隔或固定危险品。  
4.46 由于乘员、行李、邮件、货物、压舱物等的重量、装载舱位与固定等原因，导致超过最后一分钟修正限值，航空器滑出。  
4.47 非运行阶段且非机场活动区内，航空器出现烟雾、火警等情况（假信号除外）。

类似上述条款的其他信息。

# 附件四

# 安全红线标准

# 一、局方安全红线标准

1.低于最低天气标准违规继续进近飞越最后进近定位点。  
2. 在决断高度（DA）或最低下降高度（MDA）以下失去目视参考时，未立即执行复飞程序。

# 二、公司安全红线标准

1. 着陆跑道接地区 150 米（500 英尺）以下，未建立着陆形态并强行落地。  
2. 航前酒精检测超标。  
3. 飞行中，允许无关人员进入驾驶舱。  
4. 运行阶段，在航空器内吸烟（包括吸电子烟）。

对于触碰第 1-3 条红线的飞行机组成员的处理：取消机长现任技术岗位聘用资格 36 个月；延长副驾驶转升 36 个月。

对于触碰第4条红线的飞行机组成员，暂停派遣不少于12个月。

# 附件七

# 《民用航空安全信息管理规定》

# 第七章 法律责任

# 一、企事业单位有下列行为之一的，由局方给予警告，并责令限期改正，逾期不改正的，处3万元以下的罚款：

（一）违反本规定第五条，未按要求制定民用航空安全信息程序和机制的；  
（二）违反本规定第十一条，配备的民用航空安全信息管理人员不符合相关条件的或数量不满足工作需要的；  
（三）违反本规定第十二条，未配备民用航空安全信息管理人员必需设备，或配备的设备无法正常使用的；  
（四）违反本规定第三十二条，未建立民用航空安全信息分析和发布制度的；

# 二、企事业单位有下列行为之一的，由局方给予警告，或处1万元的罚款；情节严重的，处2万元以上3万元以下的罚款：

(一) 违反本规定第十条，未按规定报告事件信息的；  
(二) 违反本规定第十四条, 未按规定报告在我国境内发生的事件的;  
(三) 违反本规定第十五条, 未按规定报告在境外发生的事件的;

（四）违反本规定第十六条，未按规定对已上报的事件进行处理的；  
（五）违反本规定第十七条，未按规定途径上报民用航空安全信息报告表的；

（六）违反本规定第二十条，未妥善保护与事故、事故征候、一般事件以及举报事件有关的所有文本、影音、数据以及其他资料的；  
（七）违反本规定第二十三条，未遵守地区管理局制定的民用航空安全信息管理办法中综合信息和监察信息的相关要求；

（八）违反本规定第二十九条，未按规定保护举报人合法权益的；  
（九）违反本规定第三十四条，未按要求定期分析民用航空安全信息的；  
（十）违反本规定第三十七条，未按规定开展安全警示、预警工作的。

# 三、个人有下列行为之一的，由局方处1000元以下的罚款：

(一) 违反本规定第十条，未按规定报告事件信息的；  
（二）违反本规定第二十条，未妥善保护与事故、事故征候、一般事件以及举报事件有关的所有文本、影音、数据以及其他资料的；  
（三）违反本规定第二十九条，未按规定保护举报人合法权益的。
