# 飞行计算页面设计优化方案

## 设计理念

基于对项目整体设计风格的分析，我们将飞行计算页面从原来的蓝色主题调整为与其他页面一致的优雅设计风格。

## 主要改进

### 1. 页面背景
- **原设计**：深蓝色渐变背景（#3b82f6 → #1d4ed8）
- **新设计**：淡色渐变背景（#f8fafc → #f1f5f9），与首页保持一致

### 2. 标题区域
- **原设计**：蓝色背景，带发光动画效果
- **新设计**：紫色渐变背景（#667eea → #764ba2），与资料查询、航班运行页面统一

### 3. 功能卡片
- **原设计**：半透明玻璃质感，蓝色阴影
- **新设计**：
  - 纯白色背景，更加干净简洁
  - 柔和的阴影效果（0 4rpx 24rpx rgba(0, 0, 0, 0.06)）
  - 细微的边框（1rpx solid rgba(0, 0, 0, 0.05)）

### 4. 模块图标
- **新增设计**：
  - 图标放置在淡紫色渐变背景容器中
  - 96rpx × 96rpx 的圆角方形容器
  - 双层背景效果，增加视觉层次

### 5. 积分标签
- **位置调整**：从左下角移至右上角
- **样式优化**：
  - 使用渐变背景替代纯色
  - 缩小尺寸，更加精致
  - 不同类型使用不同的渐变色

### 6. 交互效果
- **点击反馈**：缩放效果（scale: 0.98）配合紫色边框高亮
- **扫光动画**：保留但调整为更柔和的白色扫光效果

## 视觉层次结构

1. **第一层级**：页面标题（紫色渐变背景）
2. **第二层级**：分区标题（黑色文字，32rpx）
3. **第三层级**：功能卡片（白色卡片，柔和阴影）
4. **第四层级**：卡片内容（图标、标题、描述）

## 色彩方案

- **主色调**：紫色渐变（#667eea → #764ba2）
- **背景色**：淡灰渐变（#f8fafc → #f1f5f9）
- **卡片色**：纯白（#ffffff）
- **文字色**：
  - 标题：#1a1a1a
  - 描述：#6b7280
- **积分标签色**：
  - 免费：绿色渐变（#10b981 → #059669）
  - 1积分：灰色渐变（#6b7280 → #4b5563）
  - 2积分：紫色渐变（#667eea → #764ba2）

## 设计效果

新设计实现了以下目标：

1. **风格统一**：与首页、资料查询、航班运行等页面保持一致
2. **视觉优雅**：采用柔和的色彩和阴影，营造高端感
3. **层次清晰**：通过颜色、大小、间距建立清晰的信息层级
4. **交互友好**：保留适度的动画和反馈效果

## 实施细节

所有样式修改集中在 `miniprogram/pages/flight-calculator/index.wxss` 文件中，主要包括：

- 容器背景调整
- 标题区域重设计
- 卡片样式优化
- 图标容器新增
- 积分标签重定位
- 交互效果优化

此设计方案确保飞行计算页面与整个应用的设计语言保持高度一致，提供统一且优雅的用户体验。
