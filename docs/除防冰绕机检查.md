# 1.外部检查项目
 
# 1.1 在地面结冰条件下，飞机起飞前必须完成以下外部检查内容：
 
 
 
A. 机翼、翼尖小翼、垂直/水平安定面和操纵面必须没有冰、雪或霜。
 
注意：如因浸冷效应（低温燃油、外界大气温度冰点以上和高湿度）引起机翼下方燃油箱区域附有小于3mm厚的霜，则不影响起飞性能。（THS不适用）
 
注意：透明冰是很难看到的,需要用手触摸以确保没有透明冰附着在飞机上。飞机表面的雪或半融雪下面会有透明冰层形成。机翼上、下翼面上的燃油箱区域内等容易产生透明冰。
 
B. 皮托管、静压孔、迎角探测器、总温传感器、结冰探头、电子舱机外通风口必须没有冰、雪或霜。
 
C. 发动机进气道、尾喷口、风扇叶片（包括风扇叶片背面）必须没有冰或雪，发动机风扇叶片能自由转动。
 
D. 冲压空气进/出口必须没有冰、雪或霜,外流活门上必须没有冰、雪或霜和障碍物。
 
E. 起落架和起落架舱门上必须没有冰、雪或霜和障碍物。
 
F. 油箱通气口必须没有冰、雪或霜。
 
G. 机身上必须没有附着的冰或雪, 机身上表面允许存在薄白霜。
 
H. 如果在上述步骤中发现冰、雪或霜,完成除冰/防冰工作。
