# CCAR规章卡片UI/UX美化说明

## 设计理念

本次美化遵循Apple设计语言,采用iOS Human Interface Guidelines的设计原则,实现了**简洁、优雅、舒适**的视觉体验。

## 主要改进

### 1. 设计系统升级

#### 色彩系统
- **主色调**: iOS蓝 `#007AFF` (替代原有的 `#1989fa`)
- **功能色系**:
  - 成功: `#34C759` (绿色)
  - 警告: `#FF9500` (橙色)
  - 危险: `#FF3B30` (红色)
  - 信息: `#5856D6` (紫色)
- **文本层级**: 采用iOS标准文本色系 (Primary, Secondary, Tertiary, Quaternary)
- **背景色**: iOS System Background `#F2F2F7`

#### 圆角规范
- Large: `28rpx` - 用于卡片主体
- Medium: `20rpx` - 用于按钮和小容器
- Small: `12rpx` - 用于标签
- Pill: `999rpx` - 用于胶囊形状

#### 阴影系统
- Small: 轻微阴影,用于次要元素
- Medium: 标准卡片阴影
- Large: 强调阴影,用于浮层
- Float: 悬浮效果阴影,用于搜索框等

### 2. 规章列表页面 (regulations/index)

#### 头部区域
- 增加渐变高光效果
- 更大的圆角半径 (`50rpx`)
- 优化标题字重和间距
- 增强文本阴影效果

#### 筛选器
- 采用iOS Segmented Control设计
- 玻璃拟态效果 (backdrop-filter)
- 活跃状态带缩放动画
- 内发光和边框高光

#### 卡片设计
- 增加左侧蓝色强调条 (按压时显示)
- 优化阴影层次
- 圆角从 `20rpx` 升级到 `28rpx`
- 添加细腻的边框

#### 状态标签
- 有效: 绿色渐变 + 阴影光晕
- 废止: 红色渐变 + 阴影光晕
- 失效: 灰色渐变
- 字号和内边距优化

#### 操作按钮
- 更大的点击区域 (`88rpx`)
- 边框增强视觉层次
- 按压反馈动画

### 3. 规章分类页面 (categories/index)

#### 搜索框
- iOS风格圆角胶囊
- 玻璃拟态背景
- 悬浮阴影效果
- 增强点击反馈

#### 统计卡片
- iOS Segmented Control风格的状态切换
- 数字采用SF Pro Display字体
- 活跃状态带缩放和阴影
- 时间筛选按钮采用紫色主题

#### 下拉菜单
- Apple Sheet风格
- 背景模糊效果 (backdrop-filter)
- 弹出缩放动画
- 选项边框优化

#### 分类卡片
- 图标容器增加渐变背景
- 添加闪光动画效果
- 左侧彩色强调条 (按压时显示)
- 统计标签采用不同颜色区分 (规章/文件)

#### 搜索结果卡片
- 统一的卡片风格
- 标签颜色语义化:
  - 规章编号: 蓝色
  - 日期: 橙色
  - 单位: 灰色

#### 日期选择面板
- 输入框采用iOS风格
- 选中状态带蓝色边框
- 快捷标签胶囊设计

### 4. 颜色语义化

不同功能采用不同颜色进行区分:

| 功能 | 颜色 | 用途 |
|------|------|------|
| 规章 | 蓝色 `#007AFF` | 主要内容、链接 |
| 规范性文件 | 绿色 `#34C759` | 文件数量、有效标签 |
| 时间筛选 | 紫色 `#5856D6` | 时间相关功能 |
| 日期 | 橙色 `#FF9500` | 发布日期标签 |
| 危险操作 | 红色 `#FF3B30` | 废止、删除 |

### 5. 交互细节

- **微交互**: 所有可点击元素添加 `0.3s cubic-bezier(0.4, 0, 0.2, 1)` 过渡
- **按压反馈**: 卡片按压时缩小到 `0.97-0.98`
- **视觉层级**: 通过阴影、边框、颜色建立清晰的视觉层级
- **字体优化**: 
  - 标题使用 `font-weight: 600-700`
  - 正文使用 `font-weight: 500`
  - 数字使用SF Pro Display字体
  - 增加 `letter-spacing` 提升可读性

### 6. 响应式适配

- 所有尺寸使用 `rpx` 单位,自动适配不同屏幕
- 内边距和间距适度增加,提升手机端点击体验
- 卡片最小高度和间距优化,避免拥挤
- 字号层级清晰,确保可读性

## 设计亮点

1. **玻璃拟态设计**: 搜索框和筛选器使用 `backdrop-filter: blur(20px)` 实现
2. **渐变光晕**: 头部背景添加径向渐变高光,增加空间感
3. **状态可视化**: 通过颜色渐变和阴影光晕强化状态感知
4. **微动画**: 卡片左侧强调条、图标闪光等细微动画提升精致感
5. **层次分明**: 通过阴影大小和边框透明度建立视觉层级

## 浏览器兼容性

- 使用 `-webkit-backdrop-filter` 兼容iOS Safari
- 所有动画使用 `transform` 和 `opacity` 实现硬件加速
- 渐变和阴影采用标准CSS3语法
