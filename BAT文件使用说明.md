# Cursor 缓存清除 BAT 文件使用说明

## 📁 文件说明

我为您创建了两个批处理文件，可以根据需要选择使用：

## 🚀 使用方法

### 方式一：右键管理员运行（推荐）
1. 右键点击 `.bat` 文件
2. 选择 **"以管理员身份运行"**
3. 按提示操作即可

### 方式二：双击直接运行
- 双击 `一键清除Cursor缓存.bat` 即可直接运行
- 如果权限不够会自动提示

### 方式三：命令行运行
```cmd
:: 运行完整版
"清除Cursor缓存.bat"

:: 运行极简版
"一键清除Cursor缓存.bat"
```

## 📋 故障排除

### 问题1：提示需要管理员权限
**解决方法**：
- 右键点击文件 → "以管理员身份运行"
- 或者在管理员命令提示符中运行

### 问题2：无法找到 Cursor 进程
**说明**：
- 这是正常的，表示 Cursor 没有运行
- 脚本会继续清除缓存文件

### 问题3：某些文件无法删除
**解决方法**：
- 确保 Cursor 完全关闭
- 检查是否有其他程序占用文件
- 重启系统后再运行

### 问题4：清除后扩展不见了
**说明**：
- 如果使用完整版并选择清除扩展，需要重新安装
- 推荐使用极简版，会保留扩展
